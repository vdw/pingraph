Quick Start

Up and running in under a minute.

bash

$ docker pull ghcr.io/vdw/pingraph:latest

$ docker run -d \

--cap-add NET_RAW \

-p 3000:3000 \

-v pingraph_data:/data/app/storage \

ghcr.io/vdw/pingraph:latest

✓ Open http://localhost:3000 — sign in and start monitoring.

Needs NET_RAW capability to send ICMP packets. See the Docker docs for compose examples.
